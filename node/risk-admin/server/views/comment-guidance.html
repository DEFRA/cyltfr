Guidance for writing content on GOV.UK

  • Use plain English. Read words to avoid
  • Write short clear sentences
  • You can use technical terms but you must explain them
  • Avoid abbreviations and acronyms
  • Do not write blocks of text in capitals
  • Write dates like this: 4 June 2017
  • Use ‘to’ in date ranges rather than -
  • Avoid negative contractions
  • Avoid ampersands, use ‘and’
  • Numbers: generally use numerals but there are exceptions

For further guidance read:

  • Writing for GOV.UK
  • Government Digital Service style guide
