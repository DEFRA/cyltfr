# LTFRI 2.1.9 Wednesday 25th May 2022

# Instructions

This release is a data update of the Target Area and ROFRS data sets and also a refresh of the ROFRS tile cache.

Execute jobs:

LTF_{stage}_99_UPDATE_TARGET_AREAS (5 mins)

LTF_{stage}_99_UPDATE_ROFRS (5 hours)

LTF_{stage}_99_COPY_CACHE (1 hour)

 -- Select ROFRS (Risk of flooding from Rivers and Sea) checkbox

A full code build also needs to be performed as normal to update the LTF-APP and LTF-SERVICE application version numbers.
